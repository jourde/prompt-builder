I am going to give you a spreadsheet containing responses collected from a group of 
educational support coordinators in European Schools. They were asked to describe their 
professional needs in order to adapt a prompt-building tool to their work.

══════════════════════════════════
YOUR TASK
══════════════════════════════════

Analyse all responses and produce a clean, consolidated list organised under the four 
headings below. For each heading, follow the three-step process.

Headings:
```
  # PERSONA / ROLE
  # OBJECTIVE
  # AUDIENCE
  # BOUNDARIES & STYLE
```

Three-step process for each heading:

  STEP 1 — MERGE
  Identify responses that express the same idea, even if worded differently.
  Merge them into a single item using clear, neutral professional language.

  STEP 2 — STANDARDISE
  Write each item as a short, self-contained phrase (not a full sentence).
  Use consistent grammatical form across all items under the same heading
  (e.g. all noun phrases, or all verb phrases beginning with an infinitive).

  STEP 3 — ORGANISE
  Under each heading, sort items from most to least frequently mentioned.
  Flag items that appeared only once with the label [unique].
  Flag items that seem ambiguous or unclear with the label [to clarify].

══════════════════════════════════
OUTPUT FORMAT
══════════════════════════════════

For each heading, produce two outputs:

A) CONSOLIDATED LIST
   A numbered list of all merged, standardised items.
   Include frequency count in brackets after each item, e.g. (×4).

B) INTERFACE RECOMMENDATION
   For each item, recommend one of the following input types for the interface:
     - FREE TEXT     → open-ended, too varied for a fixed list
     - DROP-DOWN     → a small fixed set of options, mutually exclusive
     - TICK BOXES    → multiple values can apply simultaneously
   
   Present this as a small table with three columns:
   Item | Recommended input type | Reason (one sentence)

══════════════════════════════════
IMPORTANT CONSTRAINTS
══════════════════════════════════

- Do not invent items. Every item in your output must trace back to at least one response 
  in the spreadsheet.
- Do not discard any response without explanation. If a response does not fit under any 
  heading, collect it in a final section called # UNCLASSIFIED and explain briefly why.
- Preserve the voice of the respondents where possible. Standardise wording, but do not 
  replace professional terminology with generic language.
- If the spreadsheet contains responses in more than one language, work from the meaning, 
  not the wording. Produce all output in English.
